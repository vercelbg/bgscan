# ابزار bgscan

برنامه **bgscan** یک اسکنر چندپروتکله و قدرتمند مبتنی بر ترمینال است که با زبان Go توسعه یافته است. این ابزار به شما اجازه می‌دهد تا آی‌پی‌ها و رنج‌های شبکه‌ای را با استفاده از پروتکل‌های ICMP، TCP، DNS، HTTP (شامل HTTP/3) و Xray، و تماماً از طریق یک محیط ترمینالی (TUI) جذاب و مبتنی بر کیبورد اسکن و ارزیابی کنید.

چه به دنبال تست اتصال شبکه و پیدا کردن بهترین اندپوینت‌ها باشید و چه بخواهید پروکسی‌های Xray را اسکن کنید، bgscan نتایجی سریع و قابل اعتماد را با کمترین میزان مصرف منابع سیستم در اختیارتان می‌گذارد.
