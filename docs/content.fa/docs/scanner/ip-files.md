# فایل‌های آی‌پی (IP Files)

اسکنرها آدرس‌های هدف را از روی فایل‌های لیست آی‌پی می‌خوانند. قبل از اینکه بتوانید از یک لیست برای اسکن استفاده کنید، باید آن را از طریق محیط TUI وارد (Import) کنید تا برنامه آن را اعتبارسنجی کرده و به فرمت داخلی bgscan تبدیل کند.

---

## آماده‌سازی فایل برای وارد کردن

فایلی که وارد می‌کنید باید یک فایل متنی ساده با پسوند `.txt` باشد که **در هر خط آن فقط یک آیتم** قرار گرفته باشد. در حال حاضر فقط از **IPv4** پشتیبانی می‌شود؛ آدرس‌ها و پیشوندهای IPv6 قابل قبول نیستند و هنگام وارد کردن نادیده گرفته می‌شوند.

| فرمت | مثال |
|---|---|
| یک آدرس IPv4 تکی | `192.168.1.1` |
| یک رنج IPv4 CIDR | `10.0.0.0/24` |

**قوانین:**

- در هر خط فقط یک آیتم؛ چند آدرس را در یک خط قرار ندهید.
- خطوط خالی و خطوطی که به عنوان یک آدرس IPv4 یا CIDR معتبر تشخیص داده نشوند، بدون سروصدا نادیده گرفته می‌شوند.
- ساختاری برای کامنت‌گذاری وجود ندارد؛ خطوطی که با `#` شروع می‌شوند به دلیل عدم امکان پردازش، نادیده گرفته خواهند شد.
- فاصله‌های خالی (Whitespace) ابتدا و انتهای هر خط به‌طور خودکار حذف (Trim) می‌شوند.

#### نمونه یک فایل ورودی

```text
192.168.1.1
192.168.1.2
10.0.0.0/24
172.16.50.100
203.0.113.0/28

```

## لیست‌های آی‌پی پیش‌فرض

ابزار bgscan همراه با مجموعه‌ای از لیست‌های آی‌پی پیش‌فرض در پوشه `ips/` ارائه می‌شود. این لیست‌ها بدون نیاز به هیچ مرحله واردکردنی، آماده استفاده هستند:

| نام | توضیحات |
|---|---|
| `akamai` | رنج‌های IPv4 سی‌دی‌ان Akamai |
| `aws` | رنج‌های IPv4 سرویس‌های آمازون (AWS) |
| `azure` | رنج‌های IPv4 مایکروسافت دیتاسنتر Azure |
| `bunny` | رنج‌های IPv4 سی‌دی‌ان Bunny |
| `cloudflare` | رنج‌های IPv4 کلودفلر |
| `fastly` | رنج‌های IPv4 سی‌دی‌ان Fastly |
| `gcore` | رنج‌های IPv4 سی‌دی‌ان G-Core Labs |
| `google` | رنج‌های IPv4 گوگل |
| `iran` | آدرس‌های IPv4 ایران |

این فایل‌ها از همان فرمت داخلی CSV بقیه لیست‌ها پیروی می‌کنند و می‌توانید مثل بقیه فایل‌ها نام آن‌ها را تغییر داده یا حذفشان کنید.

## مدیریت لیست‌های آی‌پی

مدیریت لیست آی‌پی را از مسیر **Main Menu → IP List** باز کنید.

در جدول نمایش داده شده، تمام فایل‌های وارد شده بر اساس جدیدترین‌ها مرتب شده‌اند و نام، تاریخ وارد کردن و حجم فایل را نشان می‌دهند. سه اکشن با کیبورد در این صفحه در دسترس است:

| کلید | اکشن |
|---|---|
| `a` | افزودن (Add) — وارد کردن یک فایل `.txt` جدید |
| `r` | تغییر نام (Rename) — تغییر نام فایل انتخاب‌شده |
| `x` | حذف (Delete) — پاک کردن دائمی فایل انتخاب‌شده |

{{< img "/bgscan-iplist.webp" "bgscan iplist" >}}

#### افزودن فایل (`a`)

۱. کلید `a` را فشار دهید. یک پنجره انتخاب فایل (File picker) باز می‌شود.
۲. فایل `.txt` خود را پیدا و انتخاب کنید.
۳. کادری برای وارد کردن نام ظاهر می‌شود. یک نام برای این لیست بنویسید (مثلاً `internal-servers`).

- نام وارد شده باید یک نام مجاز برای فایل در سیستم‌عامل باشد.
- **نام باید منحصربه‌فرد باشد** — نمی‌توانید از نامی که در حال حاضر وجود دارد استفاده کنید.
۴. کلید Enter را برای تایید بزنید. bgscan فایل را کپی کرده و با تبدیل فرمت، آن را در پوشه `ips/` ذخیره می‌کند.

هنگام وارد کردن، تک‌تک خطوط پردازش و استانداردسازی می‌شوند. خطوط نامعتبر بدون ایجاد خطا در کل فرآیند، خودبه‌خود کنار گذاشته می‌شوند.

{{< img "/bgscan-select-iplist.webp" "bgscan select iplist" >}}

#### تغییر نام فایل (`r`)

یک فایل را از جدول انتخاب کرده و کلید `r` را بزنید. نام جدید را وارد کرده و Enter را فشار دهید. نام فایل روی دیسک بلافاصله تغییر می‌کند.

<blockquote>نام جدید هم باید منحصربه‌فرد باشد — نمی‌توانید نام فایل را به نامی که در حال حاضر استفاده می‌شود تغییر دهید.</blockquote>

#### حذف فایل (`x`)

فایل مورد نظر را انتخاب کرده و کلید `x` را بزنید. فایل برای همیشه از پوشه `ips/` حذف خواهد شد. این کار قابل بازگشت نیست.

## فرمت ذخیره‌سازی داخلی

ابزار bgscan پس از وارد کردن لیست‌ها، آن‌ها را به صورت فایل‌های CSV در پوشه `ips/` در کنار فایل اجرایی برنامه ذخیره می‌کند:

```text
/
└── ips/
├── internal-servers.csv
├── cloudflare.csv
└── ...

```

هر فایل `.csv` از یک فرمت دو ستونه استفاده می‌کند:

```text
<ip_or_cidr>,

```

- مقدار `enable` می‌تواند `1` (فعال) یا `0` (غیرفعال) باشد. آیتم‌هایی که از یک فایل `.txt` وارد می‌کنید همیشه مقدارشان `1` خواهد بود.
- اگر ستون enable وجود نداشته باشد، bgscan آن آیتم را فعال در نظر می‌گیرد.
- نیازی نیست این فایل‌ها را به صورت دستی ویرایش کنید؛ محیط TUI خودش تمام این کارها را برای شما مدیریت می‌کند.

نامی که در منوهای برنامه برای هر لیست نشان داده می‌شود، همان نام فایل بدون پسوند `.csv` است.
