# مدیریت تنظیمات در برنامه (Settings Inspector)

هنگامی که هر یک از بخش‌های تنظیمات را از منوی Settings داخل برنامه باز می‌کنید، پنجره مدیریت تنظیمات را مشاهده خواهید کرد. این پنجره شامل لیستی از فیلدهای تب‌بندی‌شده (برتبه‌بندی‌شده) همراه با عناوین، مقادیر فعلی و امکانات ویرایش است.

{{< img "/bgscan-settings.webp" "settings menu" >}}

- فیلدها را انتخاب کرده و کلید `Enter` را فشار دهید تا مقادیر آن‌ها تغییر کند.
- کلید `d` را فشار دهید تا توضیحات یا متن راهنمای مربوط به فیلد انتخاب‌شده نمایش داده شود.
- مقادیر بله/خیر (Booleans) به صورت دکمه‌های سوئیچ (Toggle) نمایش داده می‌شوند.
- برخی از پروتکل‌ها، فیلدهای اضافی را تنها زمانی باز می‌کنند که حالت (Mode) مربوط به آن‌ها انتخاب شده باشد.

تغییرات بلافاصله روی دیسک ذخیره می‌شوند و نیازی به راه‌اندازی مجدد اسکنر نیست.
