# bgscan — راهنمای کاربر

<div dir="rtl">

bgscan یک ابزار ترمینالی برای اسکن لیست‌ها و رنج‌های آی‌پی با استفاده از
انواع مختلف اسکن (ICMP، TCP، DNS، Xray) و بررسی نتایج آن است.

این راهنما به بررسی منوی اصلی و نحوه استفاده از هر بخش می‌پردازد.

## منوی اصلی

برنامه به پنج بخش تقسیم شده است:

- **منو Run Scan** — شروع یک اسکن جدید
- **منو IP List** — مدیریت لیست‌ها / رنج‌های آی‌پی که اسکن می‌کنید
- **منو ResultList** — مشاهده نتایج اسکن‌های قبلی
- **منو Xray Outbounds** — مدیریت کانفیگ‌های خروجی Xray (مورد استفاده در نوع اسکن Xray)
- **منو Logs** — مشاهده لاگ‌های رابط کاربری، هسته و دیباگ

## منو Run Scan

برای اجرای یک اسکن:

۱. یک **لیست آی‌پی** را به‌عنوان منبع اسکن انتخاب کنید. این می‌تواند:
   - لیست آی‌پی‌ای باشد که قبلاً اضافه کرده‌اید (به [منو IP List](#منو-ip-list) مراجعه کنید)، یا
   - یک فایل نتیجه از اسکن قبلی (به [منو ResultList](#منو-resultlist) مراجعه کنید)

۲. یک **نوع اسکن** انتخاب کنید: `icmp`، `tcp`، `dns` یا `xray`.
   - اگر `xray` را انتخاب کنید، باید یک **خروجی (outbound)** نیز انتخاب کنید
     (به [منو Xray Outbounds](#منو-xray-outbounds) مراجعه کنید).

۳. اسکن را شروع کنید.

## منو IP List

این منو لیست‌های آی‌پی موجود برای اسکن را نشان می‌دهد. لیست‌های داخلی شامل:

```
akamai_range_IPv4.csv  azure_range_IPv4.csv  cloudflare_range_IPv4.csv
aws_range_IPv4.csv     bunny_range_IPv4.csv  fastly_range_IPv4.csv
gcore_range_IPv4.csv   google_range_IPv4.csv iran_range_IPv4.csv
```

> **توجه:** در حال حاضر فقط IPv4 پشتیبانی می‌شود. پشتیبانی از IPv6 در آینده اضافه خواهد شد.

### مدیریت لیست‌ها

| کلید | عملکرد |
|-----|--------|
| `r` | تغییر نام لیست انتخاب‌شده |
| `x` | حذف لیست انتخاب‌شده |
| `a` | افزودن لیست جدید از یک فایل متنی |

### افزودن لیست خودتان

هنگام افزودن یک لیست، فایل باید در هر سطر یک آی‌پی یا یک رنج آی‌پی داشته باشد.
می‌توانید آی‌پی‌های تکی و رنج‌های CIDR را در یک فایل با هم ترکیب کنید.

**مثال ۱ — آی‌پی‌های تکی:**
```
192.168.1.1
192.168.1.10
192.168.1.20
```

**مثال ۲ — رنج‌ها:**
```
192.168.1.1/24
192.168.2.1/24
```

**مثال ۳ — ترکیبی:**
```
192.168.1.1
192.168.2.1/24
```

## منو ResultList

این منو نتایج اسکن‌های شما را ذخیره می‌کند. یک فایل نتیجه را انتخاب کنید
تا آی‌پی‌های پیدا شده در آن را مشاهده کنید.

| کلید    | عملکرد |
|---------|--------|
| `Enter` | کپی کردن آی‌پی انتخاب‌شده |
| `r`     | تغییر نام فایل نتیجه انتخاب‌شده |
| `x`     | حذف فایل نتیجه انتخاب‌شده |

Xray استفاده می‌شوند. برای جزئیات فرمت Xray Outbounds و نحوه افزودن آن،
اینجا را ببینید:

- [Xray Outbounds](xray_outbounds.fa.md)

| کلید | عملکرد |
|-----|--------|
| `r` | تغییر نام outbound انتخاب‌شده |
| `x` | حذف outbound انتخاب‌شده |
| `a` | افزودن outbound جدید |

## منو Logs

سه نوع لاگ در دسترس است:

- **لاگ رابط کاربری (UI)** — رویدادهای رابط کاربری
- **لاگ هسته (Core)** — لاگ اسکنر، جایی که فعالیت اسکن ثبت می‌شود
- **لاگ دیباگ (Debug)** — مخصوص توسعه‌دهندگان

اگر می‌خواهید ببینید در حین اسکن چه اتفاقی می‌افتد، لاگ هسته را مشاهده کنید.
همچنین می‌توانید با فشردن کلید `l` در حین اسکن، لاگ اسکنر را به‌صورت زنده مشاهده کنید.
